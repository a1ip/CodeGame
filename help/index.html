Bull - Tells that the letter in the challenged word matches exactly at the same position.

Challenged Word : CODE

Guessed Word : GAME

The last letter E matches exactly at the same place. So, You will get 1 Bull, 0 Cow

Cow - Tells that the letter in the challenged word is present in the guessed word. But that letter is in wrong position.

Challenged Word : CODE

Guessed Word : DAWN

The last letter D matches, but it is in wrong position. So, You will get 0 Bull, 1 Cow

You can get confused when you get complicated words having repeating letters.

For Example, if the Challenged word is, DELL and you guessed LINK, then you will get 0 Bull, 2 Cow. The 1st Cow is for the L in LINK matching with the 3rd positioned L in DELL. The 2nd Cow is for the L in LINK matching with the 4th positioned L in DELL.

Coins - You will gain coins when you win. You will get Coins based on the number chances you used to guess the word. If you win a game you will be awarded a minimum of 5 coins. If you lose a game, you will lose 2 coins. You can use Coins to add more chances for a game.
